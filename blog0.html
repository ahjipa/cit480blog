Basics of HTML

HTML, also known as Hypertext Markup Language, is the essential structure for a standard webpage. Combined with other programming languages such as, JavaScript and Cascading Style Sheets (CSS), one can create webpages available on the World Wide Web. These langauges allow the contents of a webpage to be presented as clearly and aesthetically at the creator's discretion.

From a basic point of view, the structure of a webpage relies heavily on HTML. On the other hand, CSS controls a page's layout and design, while JavaScript works with how it should behave. Together, they are able to create a cohesive and aesthetic webpage.

Through HTML, creators are able to cohesively label and identify parts of their page throught the use of tags. Tags allow for a better organized format when tracking content. Through this, a web developer and/or a UX designer is better able to apply attributes and formats to a page's content, which can then ultimately conclude into an average website.

Any user with access to a webpage can "inspect" what makes up a page, as seen above, easily by simply doing the following:
Right Click on any Website of Choice > Click "Inspect [Element]"
Once this the screen is similar to the photo above, drag the cursor around on the webpage and see how different elements are highlighted to correspond with their respective code.

A standard webpage consists of two things: a header and a body. First and foremost, a doctype must be declared to state what version of HTML will be reflected in the content of the code. This is important as certain versions will provide access to different elements and attributes. The header is also where a link to external files like a CSS and/or JavaScript file can be declared to apply to the rest of the content within the webpage. Within the header, the meta and title should be declared to set the encoding and provide a title for the document, respectively. On the other hand, the body is where the content of the page will be. Here visual elements such as, footer, header, paragraph, sidebars, etc, can be found.
