Docker Commands

In previous and current CIT classes, I have had the opportunity to have labs that utilize Terminal. In this week's blog, I will list common and useful Terminal commands.

Commands

cd
Changes the directory of the command line path.
ls
BLists the contents of a directory.
open
Opens specified file.
cp
Copies file into another directory.
mv
Moves/renames file into a specified directory.
mkdir
Creates a directory.
rmdir
Removes an empty directory.
rm -R
Removes nested directories.
sudo
IExecutes commands with superuser privileges.
top
Lists actively running computer processes.
q
Quits subscreen and returns to Terminal. Commonly used on man page.
clear
Clears the Terminal screen of all current commands on the window.
whatis
Retrieve description for a command.
man
Pulls up manual page for a command.
exit
Closes currently open Terminal session.
touch
Creates a new file without any extension.
chmod 755 <file>
Changes the permission of a file to 755.
chown <user>:<group> <file>
Changes the ownership of a file to user and group.
kill PID
Quits process with ID <PID>.
ping <host>
Pings host and displays connectivity status.
