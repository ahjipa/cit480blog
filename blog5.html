Online Resume

his semester, I am taking CIT384/L alongside the rest of my courses. CIT384 is a Web Developing and Hosting class taught by Professor Marin. In the class we explore internet infrastructure and networking technologies. Topics we cover include: system and software architectures for web applications and e-business systems, principles of website design, advances in web-engineering technologies, principles of web-based transaction processing, XML and the associated technologies, web service technology, security and privacy issues, and so on. In this week's blog, I will discuss our continuous assignment built around creating our own personal online resume page for CIT384.

The Lab

For these next two weeks in CIT384, we are tasked with building up an online resume utilizing HTML and CSS. From here we should be able to acquire the necessary tools to go forth with the second project for this class. Given that this particule lab is something I can use as leverage for I took the time to brainstorm possible layouts with the knowledge I already knew about basic blocking and possible techniques I can take from simple Google searches.

Dividing Sections

To start off, I decided to populate the the page with basic resume information and took the time to really personalize it. Luckily, I have been regularly updating and attempting to improve on the basic design of my actual resume that I use to apply for internships and jobs, and so it was easy to fill in the basic information required.

The sections were divided into the following:

  • Header: Photo, Name, Position
  • Education: Basic Information on Education ie. Degree, Expected Graduation
  • Professional Experiences: Relevant work experiences that can also include internships and volunteer history.
  • Footer: Contact Information, Copyright Mark

Prior to customizing the HTML to fit the content and layout I wanted, the basic layout was set up as seen above.

After filling in the needed information, I started playing around with the theme and overall look of the page. I did some font mixing and added in some neutral tones in order to keep it professional. Using Google Fonts, I imported PT Serif and Nunito Sans into my page for the Headers and default font by doing the following: 

h1, h2, h3, h4, h5, h6, p {
	font-family: 'PT Serif', serif;
  margin: 0;
	padding: 0;
}
body {
  font-family: 'Nunito Sans', sans-serif;
  width: 80%; 
	margin-right: auto; 
	margin-left: auto; 
	padding: 2.5% 2.5% 0;
	background: white; /* Sets the body background to white. */
	line-height: 1.8; /* Declares spacing vertically */
}
<link href="https://fonts.googleapis.com/css2?family=PT+Serif&display=swap" rel="stylesheet"> 
  
<link href="https://fonts.googleapis.com/css2?family=Nunito+Sans:wght@300&display=swap" rel="stylesheet">
The first two blocks were put into my CSS style sheet, while the last block utilizing "link" was included in the header of my HTML page.

Other changes I made included changing my image shape to be rounded instead of square and resizing the jpeg to fit within the page; I achieved so with the following code: 

img {
  border-radius: 50%; /* gives images rounded shape */
  max-width: 25%;
  max-height: 25%;
}

Something I had a fun time learning about was utilizing icons to link to certain pages or applications. For my page, I included icons to be redirected to my email information and LinkedIn Page.

If you click the icons above, you will be directed to their intended addresses. This can be done by doing the following: 

<script defer src="https://use.fontawesome.com/releases/v5.0.8/js/all.js"></script>
