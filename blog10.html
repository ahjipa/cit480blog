Docker Commands

In previous and current CIT classes, I have had the opportunity to have labs that utilize Docker. In this week's blog, I will list common and useful Docker Commands.

Commands

docker attach
Attach local standard input, output, and error streams to a running container.
docker build
Build an image from a Dockerfile.
docker builder
Manage build.
docker commit
Create a new image from a container's changes.
docker config
Manage Docker configuration.
docker container
Manage containers.
docker cp
Copies files and/or folders between a container and the local file system.
docker create
Creates a new container.
docker diff
Inspect changes to files or directories on a container's file system.
docker exec
Runs a command in a running container.
docker export
Exports a container's file system as a tar archive.
docker history
Shows the history of an image.
docker image
Manages images.
docker images
Lists images.
docker import
Imports the contents from a tarball to create a filesystem image.
docker info
Displays system-wide information.
docker inspect
Return low-level information on Docker objects.
docker kill
Kills one or more running containers.
docker login
MLog in to a Docker registry.
docker logout
Log out from a Docker registry.
docker logs
Fetches the logs of a container.
docker network
Manages networks.
docker pause
Pauses all processes within one or more containers.
docker port
Lists port mappings or a specific mapping for the container.
docker ps
Lists containers.
docker pull
Pull an image or a repository from a registry.
docker push
Push an image or a respository to a registry.
docker rename
Renames a container.
docker restart
Restarts one or more containers.
docker rm
FRemoves one or more containers.
docker run
Runs a command in a new container.
docker start
Start one or more stopped containers.
docker stop
Stops one or more running containers.
docker update
Update configuration of one or more containers.
docker version
Show the Docker version information in depth.
